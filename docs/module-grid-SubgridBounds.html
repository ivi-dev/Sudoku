Class: SubgridBounds

grid~SubgridBounds(row, col)

A class representing the bounds of a subgrid expressed as the grid row and column that its top left edge is located at in the containing grid. E.g., a 9x9 grid is made out of 9 3x3 subgrids. The second subgrid (top center) would then have a bounds of 0:3, meaning its top left edge is sitting on the first row and the 4th column of its containing grid.

Constructor

new SubgridBounds(row, col)

Construct a new bounds instance with the provided row and column coordinates.
Parameters:
Name Type Description
row number The subgrid's top left edge's row.
col number The subgrid's top left edge's column.
